Python Sequences¶

Lists¶

A list contains a sequence of data points, such as floats, Booleans, integers, strings, etc. We use SQUARE BRACKETS to create lists. We can also slice lists, and they are similar to vectors in R. Pandas and Numpy also have their own types of list, as well as other data objects. Lists are also similar to arrays in R, in that we can creates lists of lists, etc.

del Family[2]

Dictionaries¶

Dictionaries are another way of storing data- each value in a dictionary is associated with a key- a key and its respective value form a key-value pair. After a dictionary is created, each value can be accessed by its key. This could be useful, for example when we have a company as the key and its stock price as the value- enables to look up and return values associated with a given key

In [20]:
dict = {'Apple': 102.9, 'Amazon': 110.9, 'Facebook Wankers': -9.02}

Recall, expected portfolio returns is given by the sum of the weighted annualized log returns and the portfolio volatility is the square root of the portfolio variance, which is a combination of the variance of the returns of each stock and the covariance of the returns between the stocks in the portfolio

In [32]:
log_returns = np.log(pf_data / pf_data.shift(1))
num_assets = len(assets)

#create empty lists
pfolio_returns = []
pfolio_volatility = []

for x in range(1000):
    weights = np.random.random(num_assets)
    weights = weights / np.sum(weights)
    #Portfolio returns = weighted annualized returns for each stock
    pfolio_returns.append(np.sum(weights * log_returns.mean() * 250))
    #Portfolio volatility = square root of portfolio variance
    pfolio_volatility.append(np.sqrt(np.dot(weights.T, np.dot(log_returns.cov() * 250, weights))))
